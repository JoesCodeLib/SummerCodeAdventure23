Prototype Chain

  • Prototype enables inheriting properties and methods from other objects. Every object = 1 prototype. If more than 2, null.
  • Don't really have to know

    • Destructuring

  • Instead of "const name = dog.name;", you can "const { name, dog } = dog;"
  • Can do same for arrays; const [ , , c] = arr; will assign the third array element to c.

    • ...spread

  • When you see "..." it means two objects merging
  • ex) obj2 ={...obj1, third = 'd', fourth = 'e'}

    • Optional Chaining

  • Use question mark "person?.name" to see if object is undefined.
